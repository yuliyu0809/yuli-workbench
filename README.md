# Yuliの工作台

个人使用的 TEMU 灯饰运营工具网站，基于 React、Vite、TailwindCSS 构建。当前版本包含首页、标题工坊、提示词工坊、利润计算器、我的规则中心、产品配置中心和备份中心，数据保存在浏览器 LocalStorage。

## 运行方式

```bash
npm install
npm run dev
```

打开：

```text
http://127.0.0.1:5173
```

## V1 功能

- 首页：查看产品、标题、提示词、利润记录概览。
- 标题工坊：输入产品信息，生成多组 TEMU 英文标题，支持复制和保存。
- 提示词工坊：按产品、颜色、图片类型、平台生成中文/英文灯饰提示词，支持复制和保存。
- 利润计算器：计算单件利润、利润率、保本售价和建议售价区间。
- 我的规则中心：集中查看 TEMU 灯饰运营规则。
- 产品配置中心：新增、编辑、删除产品，并作为提示词工坊的数据源。
- 备份中心：自动保存最近 10 份本地快照，并支持下载 JSON 备份文件。

## 数据说明

所有数据都保存在当前浏览器的 LocalStorage 中，不需要服务器和数据库。更换浏览器或清理浏览器数据后，本地保存内容会受到影响。
