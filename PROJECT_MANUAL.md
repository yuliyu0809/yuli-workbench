# Yuliの工作台 项目说明书

## 项目定位

Yuliの工作台是一个本地运行的 TEMU 灯饰运营工具网站，面向个人使用，不需要服务器、不需要数据库，所有业务数据保存在浏览器 LocalStorage。

## 技术栈

- React
- Vite
- TailwindCSS
- lucide-react 图标
- LocalStorage 本地存储

## 当前版本功能

### 首页

- 展示太阳能灯饰产品、标题草稿、提示词、利润记录、运营规则数量。
- 展示最近产品：太阳能煤油灯串、太阳能足球灯串、太阳能气泡球灯串、太阳能蘑菇引路灯。
- 提供标题工坊、提示词工坊、利润计算器、规则中心的快捷入口。

### 标题工坊

- 根据英文产品名、核心关键词、卖点、场景和人群生成 TEMU 英文标题。
- 支持复制和保存标题草稿。
- 默认面向太阳能灯饰标题表达。

### 提示词工坊

- 支持产品、颜色、图片类型、平台四项核心选择。
- 产品包含：
  - 太阳能煤油灯串
  - 太阳能足球灯串
  - 太阳能气泡球灯串
  - 太阳能蘑菇引路灯
- 颜色包含：暖色、白色、彩色。
- 图片类型包含：主图、场景图、四宫格、尺寸图、SKU图。
- 平台包含：TEMU欧洲站、Amazon欧洲站。
- 同一产品不同颜色会匹配不同场景类别，避免复用相似场景。
- 输出中文提示词和英文提示词，支持一键复制和保存历史记录。

### 利润计算器

- 输入销售价、供货价、运费、广告成本、平台佣金和退款预留。
- 输出单件利润、利润率、保本售价、总成本和建议售价区间。
- 支持保存利润记录。

### 我的规则中心

预置 TEMU 灯饰运营规则：

- 同一产品不同颜色必须使用不同场景
- 标题数字放最前面
- 禁止出现 waterproof
- 统一使用 IP44
- 尺寸图采用：上半真实场景图，下半规格/尺寸/太阳能面板参数
- TEMU欧洲站优先庭院、阳台、花园、门廊场景
- 禁止出现酒、酒杯、水印、清晰人脸

### 备份中心

- 数据变化后自动生成本地备份快照。
- 最多保留最近 10 份备份。
- 支持手动立即备份。
- 支持下载备份 JSON 文件到电脑。
- 备份内容包含产品、标题、提示词、利润记录和规则。

## 运行方式

```bash
npm install
npm run dev
```

打开：

```text
http://127.0.0.1:5173
```

## 构建方式

```bash
npm run build
```

构建结果输出到 `dist` 目录。

## 数据存储说明

当前项目使用 LocalStorage 保存数据。清理浏览器数据、更换浏览器或更换电脑后，本地数据可能无法继续使用。重要资料建议在备份中心下载 JSON 文件长期保存。
